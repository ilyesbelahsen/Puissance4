- Description :
---------------
Ce projet est une implémentation numérique du jeu "Puissance 4", avec une intelligence artificielle basée sur 
l'algorithme MiniMax et l'élagage alpha-bêta. L'objectif était de créer une application interactive où les utilisateurs
peuvent jouer contre l'IA ou un autre joueur, en utilisant des technologies modernes de programmation. Ce travail a permis 
d'approfondir nos compétences en développement logiciel et en intelligence artificielle.

- Fonctionnalité : 
------------------
- Jeu Solo contre l'IA : Permet aux utilisateurs de défier une intelligence artificielle avancée qui utilise l'algorithme MiniMax 
avec élagage alpha-bêta pour optimiser ses décisions de jeu.
- Évaluation Dynamique du Plateau : L'IA évalue le plateau à l'aide d'une fonction heuristique, calculant les meilleures positions et 
mouvements pour maximiser ses chances de gagner.
- Interface Utilisateur Interactive : Propose une interface graphique claire et facile à utiliser, rendant l'expérience de jeu 
intuitive et engageante pour les joueurs.
